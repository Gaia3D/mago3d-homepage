mago3D CE/EE

mago3D는 오픈 소스 기반 디지털트윈 플랫폼입니다.
Github에서 소스를 다운받아 개발 및 상업적 용도로 사용할 수 있습니다.
커스터마이징, 성능 최적화, 기술지원 등의 부가서비스가 필요하신 경우, 상용 버전인 Enterprise Edition을 이용하실 수 있습니다.

CE(Community Edition) EE(Enterprise Edition)

F4D Converter
  • 3D 파일(citygml, indoorgml, ifc, 3ds, obj, collada(dae) 등)을 웹 브라우저가 가시화 가능한 포맷으로 변환
Base Map
  • 온프레미스 폐쇠망에서 사용할 수 있는 배경지도 제공 (NASA - Blue Marble 위성영상)
Terrain Provider
  • DEM 데이터를 통해 타일을 생성하여 3차원 지형정보를 가시화
GeowebCache
  • 레이어(배경지도, 항공영상, 위성영상 등) 성능 최적화를 위한 타일 캐싱 기능 제공
Smart Tiling
  • 스마트 타일이란 3차원 데이터를 피라미드 타일 구조로 저장하고 전체 데이터를 불러오지 않으며 화면에 들어오고 카메라와의 거리를 따진 타일의 데이터만 호출하여 3D 가시화 성능 최적화를 위해 사용
DB이중화
  • 시스템 오류로 인한 데이터베이스 서비스 중단이나 물리적 손상 발생 시 이를 복구하기 위해 동일한 데이터베이스를 복제해 관리하는 기능 제공
교육 및 기술지원
  • mago3D 설치/개발/운영에 대한 교육 및 기술지원 혜택 제공

구성요소

mago3D 는 웹 데이터 관리 시스템, 가시화 엔진 라이브러리, 데이터 변환기 시스템으로 구성되어 있습니다.

웹 데이터 관리
  • 데이터관리기능(2D/3D)
  • 다양한 시뮬레이셔 관리
  • Rule-Based Management System을 이용한 연동관리
  • 대시보드, 사용자, Role, 메뉴, 환경설정 등 관리기능 제공
가시화엔진 mago3D JS
  • 순수 웹 브라우저 기술 HTML, CSS,JAVASCRIPT를 이용한 3D 데이터 가시화 기능
  • Deferred Rendering을 이용 Cesium 글로버엔진 플러그인 형식 구동
mago3D Converter
  • IFC, 3DS, OBJ, DAE, GML, LAS, JT 등 다양한 3D파일을 웹브라우저가 가시화 할 수 있는 포맷으로 변환
  • 대용량 데이터 가시화를 위한 성능 최적화 및 속성 추출
Smart Tiler
  • 웹 브라우저에서 대용량의 이 기종 3차원 데이터 컨텐츠의 요청 건수, 용량 등을 줄여 네트워크 성능 향상을 통해 고속/고품질로 서비스하는 3차원 데이터 타일링 기법 수행
